Replugged - Regroove plugins
============================


DJ-style audio effects library and VST3/LV2 plugin, for use in Live performance.


This repo contains:
  - **Modular Effects** (`effects/`)  
    Individual, self-contained effect modules
  - **VST3/LV2 Plugin** (`plugins/`)  
    For use in DAWs like Bitwig, Mixxx, etc.
  - **Korg logue Effects** (`logue/`)  
    Minimal wrappers for minilogue xd, prologue, NTS-1
  - **Rack Modules** (`rack/`)  
    VCVRack modules for Live Performance; DJ deck, etc


## Features

  - **Distortion**  
    Analog-style saturation with drive and mix controls (this started it all)
  - **Filter**  
    Resonant low-pass filter with cutoff and resonance
  - **EQ**  
    DJ-style 3-band kill EQ (Low/Mid/High)
  - **Compressor**  
    RMS compressor with soft knee and makeup gain
  - **Delay**  
    Stereo delay with time, feedback, and mix
  - **Stereo Widen**  
  - ... and many more
